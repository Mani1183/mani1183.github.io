# 🌐 Manish Kumar – Portfolio Website

A modern, responsive personal portfolio website showcasing my skills, projects, and professional background as a **Full Stack Web Developer**.  
Built with **HTML, CSS, JavaScript, and Tailwind CSS**, and deployed using **GitHub Pages**.

---

## 🔗 Live Demo
👉 **Portfolio Website:**  
https://mani1183.github.io

---

## 📌 Features

- 🌗 **Dark / Light Mode Toggle** (with preference saved)
- 🧠 **Auto-loaded Projects from GitHub API**
- 📊 **Live GitHub Stats** (Repositories & Followers)
- ✨ **Glowing Hover Effects** for buttons and cards
- 📄 **Resume Download** (Google Drive)
- 📩 **Contact Form UI** (Frontend only)
- 🚀 **Fast & Lightweight**
- 📱 **Fully Responsive Design**
- 🌍 **GitHub Pages Ready**

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Styling:** Tailwind CSS + Custom CSS
- **Icons:** Ionicons
- **API:** GitHub REST API
- **Deployment:** GitHub Pages
